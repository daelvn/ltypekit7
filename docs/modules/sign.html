applyArguments (constructor, argl, cache)
Applies arguments to to a function and checks the types of the inputs and outputs. Curried function. It returns whatever the function is supposed to return.

Parameters:

  • constructor table Signed constructor.
  • argl table List of arguments.
  • cache table Cache to initialize the application with. Passed alongside argl.

Returns:

    ...

Raises:

(#2) Did not pass enough arguments to function. (#3,#4,#10,#11) Wrong type passed as argument. (#5,#14) Type does not start with a letter. (#6) Wrong list passed as argument. (#7) Wrong table passed as argument. (#9) Function passed does not match. (#12,#13,#19,#20) Wrong type for return value. (#15) Wrong list returned. (#16) Wrong table returned. (#18) Function returned does not match.
sign (signature, context, cache)
Signs functions.

Parameters:

  • signature string Signature to apply.
  • context table Context to sign the function with. Defaults to an empty table.
  • cache table Cache to initialize the application with. Defaults to an empty table.

Returns:

    table Signed constructor.

Usage:

    Set a function to sign with a signature, then call that placeholder with the actual function definition, and then you can use it as a normal function.
impure (f)
Makes a function "impure". Adds a ! marker to it when calling.

Parameters:

  • f function Any function.

Usage:

    printx  = impure sign "a -> Boolean"
printx! (a) -> print a
printx! "string"

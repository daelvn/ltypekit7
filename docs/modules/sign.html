sign (signature, context, cache)
Signs functions.

Parameters:

  • signature string Signature to apply.
  • context table Context to sign the function with. Defaults to an empty table.
  • cache table Cache to initialize the application with. Defaults to an empty table.

Returns:

    table Signed constructor.

Usage:

    Set a function to sign with a signature, then call that placeholder with the actual function definition, and then you can use it as a normal function.
impure (f)
Makes a function "impure". Adds a ! marker to it when calling.

Parameters:

  • f function Any function.

Usage:

    printx  = impure sign "a -> Boolean"
printx! (a) -> print a
printx! "string"
flatten (fltk)
Turns a signed function into a normal function. This will make it look like a normal function to lua, but all information is lost, and therefore makes it unable to be modified.

Parameters:

  • fltk function(ltypekit) TypeKit function

Returns:

    function Flat function.

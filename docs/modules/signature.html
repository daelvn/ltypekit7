binarize (signature, context)
Returns a Right and Left part for a function. It may also return a Context.

Parameters:

  • signature string Signature to be parsed.
  • context table Context to be applied to the tree. Defaults to an empty table.

Returns:

    table Node tree for the signature. It has a left field, right field, a context field and optionally a name field.

Raises:

Unmatching parentheses.
Expected '>'.
Unexpected '>'.
Unexpected tag.

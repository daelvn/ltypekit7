parseConstructor (type_, name, cons, dorecords, doaddrefs)
Parses a constructor out of a constructor string.

Parameters:

  • type_ string The type that should be returned by the constructor.
  • name string The name of the constructor.
  • cons string The annotation of the constructor.
  • dorecords boolean Defaults to true. Set to false if you do not want record functions to be produced.
  • doaddrefs boolean Defaults to true. Set to false if you do not want record functions to be added to _G.__ref

generateSavePairing (constructor, tail)
Associate indexes in a value of a certain type to the name of the values they should be saved under.

Parameters:

  • constructor string The constructor string that generated the value.
  • tail table Names for each of the values in positional order.

Returns:

    table The generated save pairing.

Usage:

    generateSavePairing "Point Number Number", {"_", "a"}
generateSavePairing "Point2 x:Number y:Number", {"Number", "c"}
